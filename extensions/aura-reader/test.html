Test Aura Reader Extension

Instructions:

  1. Load the extension in Chrome (chrome://extensions → Load unpacked → select extensions/aura-reader)
  2. Select any text below
  3. Press Alt+Shift+E or click the extension icon
  4. Check if the popup shows your selection and context

Sample Content:

Machine learning is a subset of artificial intelligence that enables computers to learn from data without being explicitly programmed. It uses algorithms to identify patterns and make decisions based on those patterns.

Deep learning is a specialized form of machine learning that uses neural networks with multiple layers. These networks can process vast amounts of data and extract complex features automatically.

Natural language processing (NLP) combines machine learning with linguistics to help computers understand human language. Applications include chatbots, translation services, and sentiment analysis.

Test Cases:

  • Select "Machine learning" - should capture the entire first paragraph as context
  • Select "neural networks" - should capture the second paragraph
  • Select nothing and open popup - should show warning

Expected Behavior:

When you select text and trigger the extension, it should:

  1. Capture your selected text (up to 6000 chars)
  2. Find the nearest paragraph/div/article container
  3. Extract the full text from that container as context
  4. Show both in the popup with page title and URL
  5. Allow you to ask a question and send to Aura backend

Troubleshooting:

If it's not working:

  • Check browser console (F12) for errors
  • Verify extension is loaded and enabled
  • Check if content.js is injected (look in DevTools → Sources)
  • Make sure you added placeholder icons (or extension won't load)
  • Test the backend URL: https://aura-ai-assistant.onrender.com/api/health
